**Phone number field per D-02** — add as last item inside the existing `Section("Player Info")`, after the conditional defaultMatching Picker:
```swift
TextField("Phone", text: Binding(
    get: { phoneNumber ?? "" },
    set: { phoneNumber = $0.isEmpty ? nil : $0 }
))
.keyboardType(.phonePad)
```

**Contacts section per D-01** — add a new `Section("Contacts")` AFTER the `Section("Player Info")` closing brace, still inside the `Form`:
```swift
Section("Contacts") {
    ForEach(contactIdentifiers, id: \.self) { identifier in
        Text("Linked Contact")
            .foregroundStyle(.secondary)
    }
    .onDelete { offsets in
        contactIdentifiers.remove(atOffsets: offsets)
    }
    Button {
        showContactPicker = true
    } label: {
        Label("Add Contact", systemImage: "plus")
    }
}
```

Per D-05: show "Linked Contact" placeholder text only — Phase 3 resolves real names.
Per D-03: "Add Contact" button with plus icon at bottom of section.
Per D-06: swipe-to-delete via `.onDelete`.

**Sheet presentation** — add `.sheet` modifier on the `Form` (after the closing brace of Form, before `.navigationTitle`):
```swift
.sheet(isPresented: $showContactPicker) {
    ContactPickerRepresentable { identifier in
        if !contactIdentifiers.contains(identifier) {
            contactIdentifiers.append(identifier)
        }
        showContactPicker = false
    }
}
```

Duplicate guard per RESEARCH.md Pitfall 5: `!contactIdentifiers.contains(identifier)` before append.
